# Features

-->Customer Segmentation: Visualizations mapping customer demographics, locations, and regional distribution.

-->Sales & Profit Analytics: Comprehensive insights into total profit, purchasing behaviors, and sales pipeline trends.

-->Performance Metrics: Key performance indicators (KPIs) highlighting conversion rates, customer retention, and overall pipeline health.

-->Custom UI/UX: Built with a custom Power BI theme (CY26SU05) and rich visual assets for a clean, professional layout.

# Prerequisites

-->Microsoft Power BI Desktop (Latest version recommended)

-->Access to the underlying CRM/customer dataset (e.g., Python-cleaned CSV, SQL database, or Excel file)

# Installation & Setup

-->Clone this repository or download the cutomer.pbix file directly.

-->Open Power BI Desktop.

-->Open the cutomer.pbix file.

-->If necessary, go to Transform Data > Data source settings to provide the connection credentials or file path to your specific customer dataset.

-->Allow Power BI to refresh the Star Schema data model and populate the dashboard visuals.

-->Save your progress for future use.

# Repository Structure
-->cutomer.pbix: The core Power BI file containing the data model, relationships, DAX measures, layout, and visual configurations.
